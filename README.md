## Overview
Pandas Data Analyzer is a desktop application built with Python that enables users to explore and analyze datasets through a graphical interface.
The application supports loading CSV and Excel files, performing common data analysis operations, and visualizing results using popular Python data libraries.
The project was created as a learning exercise focused on practical use of pandas, data analysis workflows, and GUI development.

## Features
- Dataset loading (CSV / Excel)
- Data exploration and preview
- Statistical summary of datasets
- Missing data inspection
- Basic data analysis operations (grouping, sorting, aggregation)
- Correlation analysis
- Data visualization
- Export results to Excel

## Technologies Used
- Python
- Pandas
- Tkinter
- Matplotlib
- Seaborn

## Installation
1. Clone the repository:
   ```sh
    git clone https://github.com/michal0002c/pandas-data-analyzer.git
   ```
2. Install dependencies:
   ```sh
    pip install pandas matplotlib seaborn
   ```
3. Run the application:
   ```sh
   python main.py
      ```
